# Tnyx - Nutrition Log App

A simple and effective way to track your daily nutrition and stay on top of your health goals. Available for both **Android** and **iOS**, Tnyx helps you log your meals, monitor your calorie intake, and manage your macronutrients with ease.

## Features

- **Meal Logging**: Easily log your breakfast, lunch, dinner, and snacks.
- **Nutritional Tracking**: Monitor your daily intake of calories, proteins, carbohydrates, and fats.
- **Progress Insights**: View your nutrition history and track your progress over time.
- **User-Friendly Interface**: A clean and intuitive design for a seamless tracking experience.

## Getting Started

This project is built using Flutter. To get started, follow these steps:

1.  **Prerequisites**: Ensure you have [Flutter](https://docs.flutter.dev/get-started/install) installed on your machine.
2.  **Clone the repo**:
    ```bash
    git clone https://github.com/your-username/tnyx.git
    ```
3.  **Install dependencies**:
    ```bash
    flutter pub get
    ```
4.  **Run the app**:
    ```bash
    flutter run
    ```
